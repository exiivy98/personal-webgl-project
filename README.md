# WebGL & Three.js Learning Projects

Personal projects created while learning WebGL 2.0 and Three.js.

## Projects

| # | Project | Description |
|---|---------|-------------|
| 01 | Colored Canvas | WebGL context, viewport, scissor test |
| 02 | Moving Rectangle | GLSL shaders, VBO/VAO, uniforms, keyboard input |
| 03 | Intersection | Circle/line drawing, intersection detection |
| 04 | Spinning | Transformation matrices, time-based animation |
| 05 | Pyramid | 3D geometry, texture mapping, arcball camera |
| 06 | Sunrise Arcball | Arcball rotation, 3D interaction |
| 07 | Shading | Phong lighting, flat/smooth shading, normals |
| 08 | Toon Shading | Non-photorealistic rendering, quantization |
| 09 | Solar System | Three.js, scene graph, orbital mechanics |

## Technologies

| Technology  | Description                                      |
| ----------- | ------------------------------------------------ |
| WebGL 2.0   | Low-level graphics API (projects 01-08)          |
| GLSL ES 3.0 | Shader language for vertex and fragment programs |
| Three.js    | High-level 3D graphics library (project 09)      |
| gl-matrix   | Matrix and vector mathematics library            |
| lil-gui     | Lightweight GUI controls                         |
